This photo gallery uploads images in the zoom mode by means of Ajax requests. When you click on a thumbnail of any image it sends Ajax requests to another web page hosted on a web server. Th web server returns a separate web page with the zoom image and then the initial web page, where you clicked the thumbnail image, displays the web page retuned from the web server in the space below.
You can download this ajax exercise from THIS LINK and take a look at it how it works.
