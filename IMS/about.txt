The current folder contains two Inventory Tracking Applications and one SQL Script file to create tables, relationships 

1) "DivaReports" is the interactive application that retrieves inventory from the company database according to different criteria specified by users. 

2) "ApplicationScan" - is the interactive application that is integrated with bar code reader. The application searches items by bar code and make the list of available and unavailable items. 

3) "InventoryTrackingsystemDB" - is the SQL Script file that contains SQL Queries to create necessary tables for listed here inventory tracking applications
